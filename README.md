# SMC Vision System (Offline)

Visual Smart Money Concept (SMC) setup validator using computer vision.

## Features
- Order Block, Liquidity, FVG detection
- A+ setup filtering
- Confidence scoring
- Auto journaling
- Web interface
- Fully offline

## Install
pip install -r requirements.txt

## Train
python train.py

## Run Web App
python app.py
Open http://localhost:5000
